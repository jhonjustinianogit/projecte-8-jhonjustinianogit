# Solució: Tasca 01: Analitzar necessitats clients

# Análisis del estado actual

La empresa Fruits de la terra es una explotación agrícola que utiliza medios anticuados. Su objetivo es conseguir la digitalización para mejorar la eficiencia, la sostenibilidad y la trazabilidad de sus productos.

Actualmente, la empresa no tiene conexión a internet, lo que les impide totalmente digitalizarlo

La gestión de las actividades agrícolas se hace con un cuaderno de papel, donde se registran de forma manual los riegos, tratamientos fitosanitarios, etc. Este sistema (ya anticuado), produce que haya más facilidad de errores humanos, la pérdida de información, etc.

En cuanto a la infraestructura tecnológica, no hay sistemas informáticos en condiciones ni de dispositivos preparados para el entorno agrícola, lo imposibilita la modernización de la empresa.

# Hardware

## Hardware para la oficina

**Objetivo**

El objetivo de la empresa en conseguir un equipo para:

- Gestión de datos
- Uso del software agrícola
- Almacenamiento y copias de seguridad

**Propuesta**

Tras analizar las necesidades del cliente hemos llegado a la conclusión de que encajaría un equipo con los siguientes componentes:

- Procesador: Intel i5 
- Ram: 16GB
- Disco: 1TB
- Sistema operativo: Windows

**Copias de Seguridad**

- Local:
  - Importante hacer copias de seguridad local en un disco externo, ya que en caso de algún fallo en el hardware, desastre natural, etc. Hay un backup para recuperar la información.

- Nube:
  - Mejor opción que la de hacer la copia de seguridad en local, ya que hoy en día son muy fiables y no hay “límite” de almacenaje.

## Hardware para el campo

**Objetivo**

Poder trabajar y registrar datos en tiempo real del campo agrícola

**Propuesta**

Móviles o tablets diseñados primordialmente para entornos duros

- **Puntos clave:**
  - Resistencia a golpes, polvo y agua
  - Pantalla visible bajo la luz del sol
  - Batería de larga duración
  - Funciona igualmente con guantes

| Opción         | Ventajas        | Desventajas   |
|----------------|-----------------|----------------|
| Rugerizado     | Muy resistente  | Más caro       |
| Tablet + funda | Más barato      | Menos duradero |

**Dependiendo de las capacidades económicas del cliente, optará por una decisión u otra.**

# Programario: Local vs Nube

## Objetivo

Decidir cuál es la mejor opción basándonos en las necesidades del cliente:

- Ubicación rural
- Problemas de conectividad
- Facilidad de uso
- Mantenimiento

## Software en local

**Ventajas:**
- No depende del internet
- Mayor control sobre los datos

**Inconvenientes:**
- Instalación más compleja
- Necesidad de mantenimiento técnico
- Acceso limitado

## Software en la nube

**Ventajas:**
- Acceso desde cualquier ubicación
- Actualizaciones automáticas
- Fácil de usar
- No requiere de mantenimiento técnico

**Inconvenientes**
- Depende totalmente del internet
- Problemas a la hora de guardar proyectos, si la conexión es mala

## Recomendación final

**Solución híbrida**

Hemos considerado que es la mejor solución para la empresa, ya que no depende siempre del internet, aprovecha las ventajas del cloud y es conveniente para entornos rurales.

# Presupuesto

## Objetivo

Calcular cuánto costará:

- Coste inicial
- Coste mensual

## Coste Inicial

**Conectividad:**

- Router 4G/5G: 120€
- Antena exterior: 150€
- Instalación y configuración: 100€

**Equipamientos de Oficina:**

- PC sobremesa: 750€
- Monitor 24”: 120€
- Disco duro externo 1TB: 80€

**DIspositivos para el campo:**

- Tablet rugerizada (Blackview): 350€X2=700€

| Concepto | Total |
| ----- | ----- |
| Conectividad | 370 € |
| Oficina | 950 € |
| Campo | 700 € |
| **TOTAL** | **2.020 €** |

## Coste Mensual

**Internet móvil rural:**

- Tarifa datos ilimitados (Movistar): 70€/mes

**Software Agrícola:**

- Agroptima / similar SaaS agrícola: 40€/usuario/mes

| Concepto | Precio |
| ----- | ----- |
| Internet | 70 € |
| Software | 40 € |
| **TOTAL** | **110 €/mes** |

# Conclusión

La digitalización de la explotación agrícola **“Fruits de la terra”** representa una oportunidad muy importante para mejorar aspectos como: La eficiencia, la sostenibilidad y la competitividad del negocio.

Los técnicos Jhon Justiniano y Hugo Muiños a partir del análisis del estado actual, han identificado algunas carencias y defectos como por ejemplo la falta de conectividad, el uso de procesos manuales y la ausencia de herramientas digitales para lo que es la gestión de la información. Para dar respuesta a estas necesidades del cliente, se ha propuesto una solución completa, la cuál incluye la implementación de una conexión a internet, la digitalización del cuaderno de campo, la incorporación de dispositivos adaptados al entorno agrícola y el uso de software adecuado.

La propuesta planteada es técnicamente viable y económicamente asumible, con una inversión inicial moderada y unos costes mensuales controlados. Además también permitirá una buena mejora significativa en lo que es la gestión de datos, la reducción de errores y una mayor trazabilidad de los productos, aspectos que cada vez son más importantes en el sector primario.

Por último, esta transformación digital no sólo optimizará los procesos actuales, sino que también preparará a la empresa para futuras innovaciones, como por ejemplo la incorporación de sensores, la automatización de riegos o el análisis avanzado de datos.

En conclusión, la implementación de este proyecto supone un paso estratégico hacia una agricultura más moderna, eficiente y sostenible.
